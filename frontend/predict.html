Méthodologie

Les prédictions sont basées sur l'historique des prix, la météo locale, les taux de change, le calendrier scolaire et les jours fériés. L'intervalle de confiance s'élargit avec l'horizon pour refléter l'incertitude. Source : API du Ministère de l'Économie (données J-1).
